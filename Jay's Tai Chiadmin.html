One-time setup to connect this app to your Google Sheet. Takes about 10 minutes on a laptop.

  1. 1
    Go to console.cloud.google.com and create a new project
  2. 2
    Search for Google Sheets API and enable it
  3. 3
    Go to Credentials → Create → API Key and copy it
  4. 4
    Create an OAuth 2.0 Client ID (Web application). Add the URL where this file is hosted as an authorised origin
  5. 5
    Paste both keys and your Sheet ID below, then tap Save

Your credentials are stored in your browser only — never sent anywhere except Google.
